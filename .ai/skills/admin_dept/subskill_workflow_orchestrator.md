# Sub-Skill: Workflow Orchestrator (The Dev Lead)
**Focus:** Managing the Development & Audit Process.

## The Master Audit Workflow:
Instrucciones para dirigir una sesión de trabajo compleja:

1.  **Phase 1: Conceptual Integrity Check**
    *   Invoca a cada Departamento (Skills) y pídeles que validen si los archivos Markdown de documentación (`.ai/brain/`) coinciden con el Código Python (`.py`).
    *   *Ejemplo:* "Skill Física, revisa si `kinetic_engine.py` realmente implementa la fórmula de Energía Cinética documentada."

2.  **Phase 2: Integration Check**
    *   Verifica los "puentes" entre departamentos.
    *   *Check:* ¿El módulo de Estructura importa y usa las validaciones de Física?
    *   *Check:* ¿El módulo de Riesgo consulta el estado de Chronos?

3.  **Phase 3: Neural Autonomy Verification**
    *   Verifica que la Arquitectura de la Red Neuronal (definida en código) tenga las entradas (Tensores) necesarias de todos los departamentos para tomar decisiones autónomas.
